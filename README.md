# Vanteo-Pay
Mobile Banking App platform for foreign nationals to access US Banking services.
## Overview

Vanteo Pay is a mobile banking platform designed to help foreign nationals access the U.S. financial system.

Users can:

- Open a U.S. bank account
- Complete identity verification
- Receive a virtual debit card
- Request a physical debit card
- Send and receive money
- Manage their finances through a secure mobile app
